ğŸ¯ About the Project
FitLife India is a single-page web application (SPA) designed to help users track their health goals while staying rooted in Indian culture. Unlike generic fitness apps, FitLife focuses on:
Dietary Context: Tracking macros for common Indian foods (e.g., Roti, Dal, Sabzi).
Traditional Exercise: Highlighting Yoga Asanas, Pranayama (breathing exercises), and Desi workouts (Dand Baithak).
Privacy: All data is stored locally in the browser (localStorage), ensuring user privacy without the need for a backend server.
